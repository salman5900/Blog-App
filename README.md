# 📝 DjangoBlog — A Minimal Blog Platform Built with Django


DjangoBlog is a fully functional blogging platform built entirely using the Django framework — backend and frontend. It leverages Django’s template system to deliver a fast, smooth experience without relying on external frontend libraries.

> 🚧 **Note**: The live website is not deployed yet. It will be launched very soon!

---

## ✨ Features

- 🖋️ Create, edit, and delete blog posts
- 🗂️ Categories or tags for organizing posts
- 🧑‍💼 Admin panel for managing content
- 🌐 Clean Django templates — no frontend frameworks
- 🖼️ Image upload support (optional)
- 📆 Automatic timestamping of posts
- 📱 Responsive and mobile-friendly design

---

## 💡 Tech Stack

- **Framework**: Django (Python)
- **Frontend**: Django Templates, HTML, CSS (vanilla)
- **Database**: SQLite (default for development)
- **Deployment**: Coming soon (Render or Railway)
